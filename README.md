# Rafael's Portfolio - Senior Full Stack Developer

Este es un portfolio profesional construido con una arquitectura moderna y de alto rendimiento.

## 🚀 Tecnologías

- **Frontend:** React (Vite), CSS Vanilla (.css separado), Framer Motion para animaciones, Lucide React para iconos.
- **Backend:** Python (FastAPI).

## 🎨 Paleta de Colores

- **Background:** `#19647E` (Teal Profundo)
- **Componentes:** `#92A4AB` (Gris Azulado)
- **Texto:** `#FFFFFF` (Blanco Puro)

## 📦 Estructura del Proyecto

- `frontend/`: Aplicación React impulsada por Vite.
- `backend/`: API construida con FastAPI de Python.

## 🛠️ Cómo Iniciar

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt # (Asegúrate de preparar requisitos)
uvicorn main:app --reload
```
